i write code and create a progam with python in where the user create and read blockers, 
in this program have a menu in where the user select options for enter opcion if that is correct to the validations


## Example usage

When you run the program, you'll see messages like:

### MENU
```text
            === MENU ===
            1. Create blocker
            2. Read blockers
            3. Exit
            Select option (1-3): 
```

### Create Blocker
```text
        === Create Daily Blocker ===
    Which is your blocker today? Database connection is down

    Blocker is saved!
```

### Read Blockers
```text
=== All blockers ===

            --- Your Blockers ---
            1. Database connection is down
            2. Waiting for API documentation
            3. Computer restart required

            Total: 3 blocker(s)
```

### Exit
```text
                === MENU ===
            1. Create blocker
            2. Read blockers
            3. Exit
            Select option (1-3): 3

            Program close.
```
